# Stufinel

Stufinel is a student budget web app that helps allocate allowance into spend, save, and invest goals.

## Features

- Choose between `Student Basic` and `Student Pro` budget modes
- Adjust category priorities with sliders
- Calculate a live allowance allocation
- View results through cards, a donut chart, and a summary snapshot

## Built With

- HTML
- CSS
- JavaScript

## Run Locally

Open [index.html](/Users/tutel/personal%20projects/stufinel/index.html) in your browser.

## Project Structure

- `index.html` - app layout
- `styles.css` - visual design and responsive styling
- `app.js` - budget logic and interactions
- `assets/` - logo and image assets
